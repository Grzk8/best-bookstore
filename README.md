# BestBookStore
BestBookStore is a webside of online bookstore.

Website was made by:
-React
-Fetch API
-React Router
-Sass

Still in progress: Redux, Authentication, leyout, other improvements...


## Instructions

To run this project, install it locally using npm:
- npm install
- npm start
- adress:  http://localhost:3000
